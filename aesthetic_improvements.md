# Aesthetic Improvements

This document captures practical visual improvements for the current landing page in `index.html`.

## Priority 1: Typography

- Replace the current mixed serif/sans feel with a more deliberate type system.
- Use one display face for headings and one readable text face for body copy.
- Increase contrast between labels, headings, and paragraph text so the hierarchy is clearer at a glance.

Why this helps:
- The page already has good structure, but the typography does not yet feel fully branded.
- A more intentional type system will make the site feel more polished without changing the content.

## Priority 2: Hero Section

- Strengthen the first-screen focal point.
- Make the headline or panel the dominant visual element, not both at the same weight.
- Reduce redundancy by replacing one of the repeated email calls to action with a stronger value statement or proof point.
- Add a subtle background treatment behind the hero panel to give the top section more depth.

Why this helps:
- The hero is the first impression.
- Small changes here will have the largest effect on perceived quality.

## Priority 3: Project Cards

- Differentiate released projects from developing projects with a visual status treatment.
- Add a status chip, border accent, or background tint to each project card.
- Make the card hierarchy more obvious by emphasizing the project name and reducing visual competition from secondary elements.

Why this helps:
- The cards are clean, but they currently feel too uniform.
- Better status styling will make the portfolio easier to scan.

## Priority 4: Section Rhythm

- Increase spacing differences between sections so the page feels less repetitive.
- Use background tints or subtle surface changes to separate major content blocks.
- Rely a little less on divider lines and a little more on whitespace.

Why this helps:
- The current page is consistent, but it needs more visual cadence.
- Stronger section rhythm makes long-form landing pages feel more designed.

## Priority 5: Buttons and Links

- Make primary actions more distinct from secondary actions.
- Use stronger contrast, slightly larger weight, or a more prominent shadow for primary CTAs.
- Keep secondary pills lighter so the interface has a clearer action hierarchy.

Why this helps:
- The page has many similar buttons, which weakens the emphasis on the most important actions.
- Clearer CTA styling improves both aesthetics and usability.

## Priority 6: Mobile Refinement

- Make the primary buttons full width or near full width on small screens.
- Preserve breathing room around logos and portraits while tightening unnecessary padding.
- Increase vertical spacing between stacked elements in the hero and project cards.

Why this helps:
- The layout is already responsive, but it can feel more intentional on smaller screens.
- Small mobile refinements usually produce a disproportionately better experience.

## Optional Direction

If a more distinctive visual identity is desired, the page could move in one of these directions:

1. Editorial and premium
2. Clean and product-focused
3. Warm and handcrafted

The current design is closest to warm and handcrafted, with some product-style structure. A small refinement pass would likely be enough to make it feel significantly more polished.
